Having Issues?

If you're encountering any sort of rendering issue with the portraits, try setting the Filter to Hard Light (toggle under title). If that fails to work in-browser, please send me a message over at Tumblr with your browser type (make sure to upgrade your browser to the latest version!) and a description and/or screenshot of the problem, and I will try to solve it to the best of my ability.

Unless you're using Internet Explorer, in which case I will just ask you to download Chrome.

If you want to use it before I push the next fix, please head over to GitHub and download a version of the source code. That will allow you to use the app locally.
